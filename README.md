#  Hospital Management System

##  What is this project?

This is a simple computer program that helps manage basic hospital work.

Instead of writing everything on paper, this system allows you to:

* Store patient details
* Store doctor information
* Book appointments

 In short, it makes hospital work **faster, easier, and organized**.

##  Why did we create this?

In many places, hospital records are still handled manually, which can cause:

* Mistakes in data
* Loss of important information
* Slow processes

This project solves these problems by keeping everything **digital**.

##  What can this system do?

###  Patient Management

* Add new patients
* View patient details

###  Doctor Management

* Add doctor information
* View doctor details

###  Appointment Booking

* Book appointments between patients and doctors
* View all appointments

## How does it work?

When you run the program, you will see a menu like this:

Hospital Management System
1. Add Patient
2. View Patients
3. Add Doctor
4. View Doctors
5. Book Appointment
6. View Appointments
7. Exit

 You just enter a number to perform an action.
